# When to use `if`


Use `if` blocks alone:
- If you want to optionally do something
- When conditions are independent
- When multiple blocks needs to run


Use `if-elif-else`
- For one choice among many
- A single decision chain
- When you only want _one_ result to happen
- I.E: Menu Choices, Grading Scores, Classifying Value 

Use `if-else`
- When there are only 2 paths `True` or `False`
- I.E: Validation, Permissions, Yes/No
